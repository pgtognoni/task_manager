This App is build with `REACT HOOKS` and `Firebase Hosting`. 
It's a RESTful application, easy to use and flexible. 
`React-Redux` is used to manage global state along with `UseContext` and `React-Router` to navigate between different views. 

To view the app go to [DX](https://doose-manager.web.app/)

Create an account to access the application.

It's a simple app to add and edit To Dos and Task Management. You can manage your agenda from the list of tasks or using the `CALENDAR`
